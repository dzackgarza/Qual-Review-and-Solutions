Sylow Theorems

A $p{\hbox{-}}$group is a group G such that every element is order pk for some k. If G is a finite $p{\hbox{-}}$group, then |G| = pj for some j.

Write

  • |G| = pkm where (p, m) = 1,
  • Sp a Sylow${\hbox{-}}p$ subgroup, and
  • np the number of Sylow${\hbox{-}}p$ subgroups.

Some useful facts:

  • Coprime order subgroups are disjoint, or more generally ℤp, ℤq ⊂ G ⟹ ℤp⋂ℤq = ℤ(p, q).

  • The Chinese Remainder theorem: (p, q) = 1 ⟹ ℤp × ℤq ≅ ℤpq

Sylow 1 (Cauchy for Prime Powers)


∀pn dividing |G| there exists a subgroup of size pn

Idea: Sylow $p{\hbox{-}}$subgroups exist for any p dividing |G|, and are maximal in the sense that every $p{\hbox{-}}$subgroup of G is contained in a Sylow $p{\hbox{-}}$subgroup.

If |G| = ∏piαi, then there exist subgroups of order piβi for every i and every 0 ≤ βi ≤ αi. In particular, Sylow $p{\hbox{-}}$subgroups always exist.

Sylow 2 (Sylows are Conjugate)

All sylow${\hbox{-}}p$ subgroups Sp are conjugate, i.e.
Spi, Spj ∈ Sylp(G) ⟹ ∃g such that gSpig − 1 = Spj

np = 1 ⇔ Sp ⊴ G

Sylow 3 (Numerical Constraints)

  1. $n_p {~\Bigm|~}m~$ (in particular, np ≤ m),

  2. np ≡ 1mod p,

  3. np = [G : NG(Sp)] where NG is the normalizer.

p does not divide np.

Every $p{\hbox{-}}$subgroup of G is contained in a Sylow $p{\hbox{-}}$subgroup.

Something proof title=“Something”

Let H ≤ G be a $p{\hbox{-}}$subgroup. If H is not properly contained in any other $p{\hbox{-}}$subgroup, it is a Sylow $p{\hbox{-}}$subgroup by definition.

Otherwise, it is contained in some $p{\hbox{-}}$subgroup H1. Inductively this yields a chain H ⊊ H1 ⊊ ⋯, and by Zorn’s lemma H ≔ ⋃iHi is maximal and thus a Sylow $p{\hbox{-}}$subgroup.

If H ⊴ G and P ∈ Sylp(G), then HNG(P) = G and [G : H] divides |NG(P)|.

Products

We have G ≅ H × K when

  • H, K ⊴ G

  • G = HK.

  • H⋂K = {e} ⊂ G

Note: can relax to [h, k] = 1 for all h, k.

We have $G = \prod_{i=1}^n H_i$ when

  • Hi ⊴ G for all i.

  • G = H1⋯Hn

  • $H_k \bigcap H_1 \cdots \widehat{H_k} \cdots H_n = \emptyset$

Note on notation: intersect Hk with the amalgam leaving out Hk.

We have G = N⋊ψH when

  • G = NH

  • N ⊴ G

  • H ↷ N by conjugation via a map
    $$ \psi: H \longrightarrow{\mathrm{Aut}}(N) \\ h \mapsto h({\,\cdot\,})h^{-1} .$$

Note relaxed conditions compared to direct product: H ⊴ G and K ≤ G to get a semidirect product instead

  • If σ ∈ Aut(H), then N⋊ψH ≅ N⋊ψ ∘ σH.

  • Aut((ℤ/(p)n) ≅ GL(n, 𝔽p), which has size
    |Aut(ℤ/(p)n)| = (pn − 1)(pn − p)⋯(pn − pn − 1).

    • If this occurs in a semidirect product, it suffices to consider similarity classes of matrices (i.e. just use canonical forms)


  • Aut(ℤ/(n)) ≅ ℤ/(n)× ≅ ℤ/(φ(n))
    where φ is the totient function.

Isomorphism Theorems

If ϕ : G → H is a group morphism then
G/ker ϕ ≅ im ϕ.

Note: for this to make sense, we also have

  • ker ϕ ⊴ G
  • im ϕ ≤ G

If ϕ : G → H is surjective then H ≅ G/ker ϕ.

If H, K ≤ G and H ≤ NG(K) (or K ⊴ G) then HK ≤ G is a subgroup.

If S ≤ G and N ⊴ G, then
$$ \frac{SN}{N} \cong \frac{S}{S\bigcap N} \quad \text{ and }\quad {\left\lvert {SN} \right\rvert} = \frac{{\left\lvert {S} \right\rvert} {\left\lvert {N} \right\rvert}}{{\left\lvert {S\bigcap N} \right\rvert}} .$$

Note: for this to make sense, we also have

  • SN ≤ G,
  • S⋂N ⊴ S,

If we relax the conditions to S, N ≤ G with S ∈ NG(N), then S⋂N ⊴ S (but is not normal in G) and the theorem still applies.

Suppose N, K ≤ G with N ⊴ G and N ⊆ K ⊆ G.

  1. If K ≤ G then K/N ≤ G/N is a subgroup
  2. If K ⊴ G then K/N ⊴ G/N.
  3. Every subgroup of G/N is of the form K/N for some such K ≤ G.
  4. Every normal subgroup of G/N is of the form K/N for some such K ⊴ G.
  5. If K ⊴ G, then we can cancel normal subgroups:
    $$ \frac{G/N}{K/N} \cong \frac{G}{K} .$$

Suppose N ⊴ G, then there exists a correspondence:


$$ \left\{ H < G {~\mathrel{\Big|}~}N \subseteq H \right\} \iff \left\{ H {~\mathrel{\Big|}~}H < \frac G N \right\} \\ \left\{{\substack{ \text{Subgroups of $G$} \\ \text{containing $N$} }}\right\} \iff \left\{{\substack{ \text{Subgroups of the } \\ \text{quotient $G/N$} }}\right\} .$$

In words, subgroups of G containing N correspond to subgroups of the quotient group G/N. This is given by the map H ↦ H/N.

Note: N ⊴ G and N ⊆ H < G ⟹ N ⊴ H.

Special Classes of Groups

The “2 out of 3 property” is satisfied by a class of groups 𝒞 iff whenever G ∈ 𝒞, then N, G/N ∈ 𝒞 for any N ⊴ G.

If |G| = pk, then G is a p-group.

If for every proper H < G, H ⊴ NG(H) is again proper, then “normalizers grow” in G.

  • Split into coprime pieces
