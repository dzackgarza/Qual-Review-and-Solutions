0.1 Sylow Theorems

A $p\dash$group is a group G such that every element is order pk for some k. If G is a finite $p\dash$group, then $\abs G = p^j$ for some j.

Write

  • $\abs{G} = p^k m$ where (p, m) = 1,
  • Sp a Sylow$\dash p$ subgroup, and
  • np the number of Sylow$\dash p$ subgroups.

Some useful facts:

  • Coprime order subgroups are disjoint, or more generally $\ZZ_p, \ZZ_q \subset G \implies \ZZ_p \intersect \ZZ_q = \ZZ_{(p,q)}$.

  • The Chinese Remainder theorem: $(p, q) = 1 \implies \ZZ_p \times \ZZ_q \cong \ZZ_{pq}$

0.1.1 Sylow 1 (Cauchy for Prime Powers)

[ p^n p^n ]

Idea: Sylow $p\dash$subgroups exist for any p dividing $\abs{G}$, and are maximal in the sense that every $p\dash$subgroup of G is contained in a Sylow $p\dash$subgroup.

If $\abs G = \prod p_i^{\alpha_i}$, then there exist subgroups of order piβi for every i and every 0 ≤ βi ≤ αi. In particular, Sylow $p\dash$subgroups always exist.

0.1.2 Sylow 2 (Sylows are Conjugate)

All sylow$\dash p$ subgroups Sp are conjugate, i.e. [ S_p^i, S_p^j _p(G) g g S_p^i g= S_p^j ]

$n_p = 1 \iff S_p \normal G$

0.1.3 Sylow 3 (Numerical Constraints)

  1. $n_p \divides m~$ (in particular, np ≤ m),

  2. np ≡ 1mod p,

  3. np = [G : NG(Sp)] where NG is the normalizer.

p does not divide np.

Every $p\dash$subgroup of G is contained in a Sylow $p\dash$subgroup.

Let H ≤ G be a $p\dash$subgroup. If H is not properly contained in any other $p\dash$subgroup, it is a Sylow $p\dash$subgroup by definition.

Otherwise, it is contained in some $p\dash$subgroup H1. Inductively this yields a chain H ⊊ H1 ⊊ ⋯, and by Zorn’s lemma $H\definedas \union_i H^i$ is maximal and thus a Sylow $p\dash$subgroup.

If $H\normal G$ and P ∈ Sylp(G), then HNG(P) = G and [G : H] divides $\abs{N_G(P)}$.

0.4 Special Classes of Groups

The “2 out of 3 property” is satisfied by a class of groups 𝒞 iff whenever G ∈ 𝒞, then N, G/N ∈ 𝒞 for any $N \normal G$.

If $\abs{G} = p^k$, then G is a p-group.

If for every proper H < G, $H\normal N_G(H)$ is again proper, then “normalizers grow” in G.
